# Data Visualisation Assignments
This repository contains assignments from my Data Visualisation class, created using Python in Jupyter Lab. Each analysis explores different datasets and provides insights through visualizations. The results are exported as HTML files for easy viewing.

# Assignments
1. GTA Online Vehicle Performance & Economy Analysis
**Objective:** Explore the relationships between vehicle price, class, and performance in GTA 5.
**Goal:** Provide insights to help players optimize their choice of vehicles.
**Techniques:** Scatter plots, boxplots, and data aggregation to analyze speed, handling, and cost.

2. Choosing the Right Pokémon: A Data-Driven Battle Analysis
**Objective:** Support players in selecting suitable Pokémon for battle by examining relationships between battle statistics and primary types.
**Goal:** Help trainers make informed choices based on strengths, weaknesses, and type advantages.
**Techniques:** Histograms, scatter plots, and type-based comparisons of HP, attack, defense, and special stats.

3. Decoding Data Science Salaries: Trends by Role, Experience, and Time
**Objective:** Explore the distribution of data science salaries and understand how compensation varies across experience levels, job roles, and over time.
**Goal:** Identify trends and disparities in data science compensation.
**Techniques:** Boxplots, histograms, and aggregation by job title and experience level.
